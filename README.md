# To-Do List App

A simple and responsive To-Do List web application built using HTML, CSS, and JavaScript.  
This project helps users manage daily tasks by allowing them to add, mark as completed, and delete tasks.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and user-friendly interface
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run the Project
1. Download or clone the repository
2. Open the `index.html` file in any modern web browser

## Project Purpose
This project is created for learning and practicing basic web development concepts using  JavaScript.
